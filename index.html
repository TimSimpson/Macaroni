Welcome!

Macaroni for C++ parses a sleeker version of C++ that resembles Java or C# and generates C++ code, so you don't have to worry about files while you're writing classes.

It also allows you to manipulating its AST and create code generators using Lua.

Macaroni is currently in a pre-alpha state and is undergoing constant changes and modifications.
